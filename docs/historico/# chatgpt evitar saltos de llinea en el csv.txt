pregunta a chatgpt ¿ como haces para que no haya saltos de linea en las preguntas y respuestas? 


Uso una normalización final de texto antes de meter cada campo en el CSV.

La idea es: dentro de cada pregunta/opción sustituyo los saltos de línea por espacios, pero sin cambiar el contenido. Así el CSV queda en una sola línea por pregunta.

import re

def limpiar_campo(texto):
    """
    Limpia un campo del CSV sin resumir ni reformular:
    - elimina saltos de línea internos;
    - convierte varios espacios en uno;
    - mantiene comas, puntos, punto y coma, signos y comillas;
    - deja el texto en una sola línea.
    """
    if texto is None:
        return ""

    texto = str(texto)

    # Unificar saltos de línea
    texto = texto.replace("\r\n", "\n").replace("\r", "\n")

    # Sustituir saltos de línea por espacios
    texto = re.sub(r"\s*\n\s*", " ", texto)

    # Reducir espacios múltiples
    texto = re.sub(r"[ \t]{2,}", " ", texto)

    # Quitar espacios al principio y al final
    texto = texto.strip()

    return texto

Luego lo aplico a cada campo antes de escribir el CSV:
